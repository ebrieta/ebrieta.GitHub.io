一些de Rham 上同调的计算例子

在这一节我们开始尝试来计算一部分de Rham 上同调群，我们最重要的计算工具仍是M—V序列。

$S^1$

我们先来计算圆周$S^1$的上同调群。
$\begin{aligned}
0 \rightarrow H^{0}\left(S^{1}\right) \rightarrow & H^{0}(U) \oplus H^{0}(V) \stackrel{\delta}{\rightarrow} H^{0}(U \cap V) \stackrel{d^{*}}{\longrightarrow} H^{1}\left(S^{1}\right)\rightarrow H^{1}(U) \oplus H^{1}(V) \rightarrow H^{1}(U \cap V) \rightarrow H^{2}\left(S^{1}\right) \rightarrow \cdots
\end{aligned}$

$R^2-P-Q$

为了计算其紧上同调群，我们先来计算$H^*(R^2-P)$,我们不妨设P为原点，我们选取$U=[(x,y)|x>0 或 y<0],U=[(x,y)|x<0,或y>0]$,则$U \coprod V=R^2-P$,由M-V序列，我们有

$\begin{array}{c}
0 \rightarrow H_{c}^{0}(U \cap V) \rightarrow H_{c}^{0}(U \coprod V) \stackrel{s}{\rightarrow} H_{c}^{0}\left(R^2-P\right) \stackrel{d_{*}}{\longrightarrow} H_{c}^{1}(U \cap V) \
\stackrel{\delta}{\rightarrow} H_{c}^{1}(U \coprod V) \stackrel{s}{\rightarrow} H_{c}^{1}\left(R^2-P\right) \rightarrow H_{c}^{2}(U \cap V) \rightarrow H_{c}^{2}(U \coprod V) \rightarrow H_{c}^{2}(R^2-P) \rightarrow \ldots
\end{array}$

在由紧上同调的poincare引理，这个正和列为：
$ 0 \rightarrow H_{c}^{0}\left(R^2-P\right)\rightarrow 0 \rightarrow 0 \rightarrow H_{c}^{1}\left(R^2-P\right) \stackrel{d *}{\longrightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{\delta}{\rightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{s}{\rightarrow} H_{c}^{2}\left(R^2-P\right) \rightarrow 0 \ \ldots$

计算紧上同调如下：
$\begin{array}{l}
H_{c}^{0}\left(R^2-P\right)=0\
H_{c}^{1}\left(R^2-P\right)=\operatorname{im} d_{*}=\text { ker } \delta=\mathbb{R} \
H_{c}^{2}\left(R^2-P\right)=\mathbb{R} \oplus \mathbb{R} / \text { ker } s=\mathbb{R} \oplus \mathbb{R} / \text { im } \delta=\mathbb{R}
\end{array}$

下面我们计算$H^*(R^2-P-Q)$，我们选取选取$U=[y<1/2]-P,V=[y>-1/2]-Q,其中P=(-1,0),Q=（1，0）$
则$U \coprod V=R^2-P-Q$,由M-V序列，我们有
$\begin{array}{c}
0 \rightarrow H_{c}^{0}(U \cap V) \rightarrow H_{c}^{0}(U \coprod V) \stackrel{s}{\rightarrow} H_{c}^{0}\left(R^2-P-Q\right) \stackrel{d_{*}}{\longrightarrow} H_{c}^{1}(U \cap V) \
\stackrel{\delta}{\rightarrow} H_{c}^{1}(U \coprod V) \stackrel{s_2}{\rightarrow} H_{c}^{1}\left(R^2-P-Q\right) \rightarrow H_{c}^{2}(U \cap V) \rightarrow H_{c}^{2}(U \coprod V) \rightarrow H_{c}^{2}(R^2-P-Q) \rightarrow \ldots
\end{array}$

在由紧上同调的poincare引理，这个正和列为：
$ 0 \rightarrow H_{c}^{0}\left(R^2-P\right)\rightarrow 0 \rightarrow \mathbb{R} \oplus \mathbb{R} \stackrel{s_1}{\rightarrow} H_{c}^{1}\left(R^2-P-Q\right) \stackrel{d *}{\longrightarrow} \mathbb{R} \stackrel{\delta}{\rightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{s_2}{\rightarrow} H_{c}^{2}\left(R^2-P-Q\right) \rightarrow 0 \ \ldots$

计算紧上同调如下：
$\begin{array}{l}
H_{c}^{0}\left(R^2-P-Q\right)=0\
再由s_1单，H_{c}^{1}\left(R^2-P-Q\right)/ \text { ker } d^*=H_{c}^{1}\left(R^2-P-Q\right)/ \mathbb{R} \oplus \mathbb{R}=\text { ker } \delta=0\
H_{c}^{1}\left(R^2-P-Q\right)=\mathbb{R} \oplus \mathbb{R} \
H_{c}^{2}\left(R^2-P-Q\right)=\mathbb{R} \oplus \mathbb{R} / \text { ker } s=\mathbb{R} \oplus \mathbb{R} / \text { im } \delta=\mathbb{R}
\end{array}$

$T^2$

下面我们计算$H^*(T^2)$，我们选取选取
$$U= \cup_n (n，n+1/2)/\mathbb{Z^2},V=\cup_n(n-2/3,n+1/3)/\mathbb{Z^2}$$
则$U \cap V=S^1 \coprod S^1,U \cup V=T^2$,由M-V序列，我们有:
$$\begin{aligned}
0 \rightarrow H^{0}\left(T^2\right) \rightarrow & H^{0}(U) \oplus H^{0}(V) \stackrel{\delta}{\rightarrow} H^{0}(U \cap V) \stackrel{d^{*}}{\longrightarrow} H^{1}\left(T^2\right) \
& \rightarrow H^{1}(U) \oplus H^{1}(V) \rightarrow H^{1}(U \cap V) \rightarrow H^{2}\left(T^2\right) \rightarrow 0\cdots
\end{aligned}$$

这个正和列为：

$0 \rightarrow H^{0}\left(T^2\right) \stackrel{s_0}{\longrightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{\delta}{\rightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{d^*}{\longrightarrow} H^{1}\left(T^2\right) \stackrel{s_1}{\longrightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{\delta}{\rightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{d^*_2}{\longrightarrow} H^{2}\left(T^2\right) \rightarrow 0 \ldots$

计算上同调如下：

$H^{0}\left(T^2\right)=\text { ker }\delta=\mathbb{R}$，$H^{0}\left(T^2\right)$代表元为常值函数。

$H^{1}\left(T^2\right)/\text { im } d^*_1=\text { ker }\delta =\mathbb{R}$

$H^{1}\left(T^2\right)=\mathbb{R} \oplus \mathbb{R} $

$H^{2}\left(T^2\right)=\mathbb{R} \oplus \mathbb{R} / \text { ker } d^*_2=\mathbb{R} \oplus \mathbb{R} / \text { im } \delta=\mathbb{R}$

$T^2-P$

注意到$T^2-P$同伦与$S^1 \wedge S^1$单点并
计算上同调如下:

$0 \rightarrow H^{0}\left(T^2-P\right) \stackrel{s_0}{\longrightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{\delta}{\rightarrow} \mathbb{R} \stackrel{d^*}{\longrightarrow} H^{1}\left(T^2\right) \stackrel{s_1}{\longrightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{\delta}{\rightarrow} 0 \stackrel{d^*_2}{\longrightarrow} H^{2}\left(T^2-P\right) \rightarrow 0 \ldots$

$H^{0}\left(T^2-P\right)=\text { ker }\delta=\mathbb{R}$

$H^{1}\left(T^2-P\right)/\text { im } d^*_1=\text { ker }\delta=\mathbb{R}\oplus \mathbb{R}$

$H^{1}\left(T^2-P\right)=\mathbb{R} \oplus \mathbb{R} $

$H^{2}(T^2-P)=0$

$S^n$

我们采用归纳法来证明，对于$S^1$我们已经知成立，现在假设n-1时已证。

$\rightarrow H^{p-1}\left(S^{n}\right) \stackrel{r}{\rightarrow} H^{p-1}(U) \oplus H^{n-1}(V) \stackrel{\delta}{\rightarrow} H^{p-1}(U \cap V) \stackrel{d^{*}}{\rightarrow} H^{p}\left(S^{n}\right) \rightarrow 0\cdots$

有$H^{p}(S^{n})=H^{p-1}(S^{n-1})$,

故有:
$$H^{q}\left(S^{n}\right)=\left
{\begin{array}{ll}\mathbb{R} & q=0,
n \ 0 & q \neq 0, n\end{array}\right.$$

###$M $

我们先来计算$H^*(M) $
选取

$\begin{aligned}
0 \rightarrow H^{0}\left(M\right) \rightarrow & H^{0}(U) \oplus H^{0}(V) \stackrel{\delta}{\rightarrow} H^{0}(U \cap V) \stackrel{d^{*}}{\longrightarrow} H^{1}\left(S^{1}\right)\rightarrow H^{1}(U) \oplus H^{1}(V) \rightarrow H^{1}(U \cap V) \rightarrow H^{2}\left(S^{1}\right) \rightarrow \cdots
\end{aligned}$

$ 0 \rightarrow H_{c}^{0}\left(M\right)\stackrel{s}{\rightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{\delta}{\rightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{d *}{\longrightarrow} H_{c}^{1}\left(M\right) \rightarrow0 \rightarrow 0 \rightarrow H_{c}^{2}\left(R^2-P\right) \rightarrow 0 \ \ldots$

$H^{0}\left(M\right)=\operatorname{ker} \delta=\mathbb{R}, \quad H^{1}\left(M\right)=\mathbb{R} \oplus \mathbb{R} / \text { ker } d^{*}=\mathbb{R}$
$H^{2}\left(M\right)=0$

我们再来计算$H^*_c(M) $

$0 \rightarrow H_{c}^{0}\left(M\right) \rightarrow 0\rightarrow 0\rightarrow H_{c}^{1}\left(S^{1}\right) \stackrel{d *}{\longrightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{\delta}{\rightarrow} \mathbb{R} \oplus \mathbb{R} \stackrel{s}{\rightarrow} H_{c}^{2}\left(M\right) \rightarrow 0 \ldots$

$H^{2}{c}\left(M\right)=\operatorname{ker} \delta=\mathbb{R}, \quad H^{1}{c}\left(M\right)=\mathbb{R} \oplus \mathbb{R} / \text { ker } d^{*}=\mathbb{R}$
$H^{0}_{c}\left(M\right)=0$
